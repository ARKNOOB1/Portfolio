# Portfolio Website

현대적이고 반응형 포트폴리오 웹사이트입니다.

## 기능

- 자기소개 섹션
- 프로젝트 경험 쇼케이스
- 이력 및 경력 정보
- 반응형 디자인
- 부드러운 애니메이션

## 실행 방법

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:3000` 접속

## 사용 기술

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Font Awesome (아이콘)
- Google Fonts
